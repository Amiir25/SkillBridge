# SkillBridge

**Skill Bridge** is a micro-internship platform that connects students with companies through short real-world projects.

---

## Key Features

- **Project Listingd**: Companies can post tasks easily.
- **Gemified Learning**: Earn XP points, badges, and climb leaderboards.
- **AI Career Guidance**: Get recommendations for skills and projects to pursue.
- **Certificates**: Showcase your real-world achievements and enhance credibility.
---

## Technologies Used

- **React**
- **Node.js**  
- **Express.js**  
- **MongoDB**  
- **Tailwind CSS**
